Recent Work

React Google Map Tweets

By leveraging the features of libraries such as google-map-react and twit I was able to create an app which appends tweet data, as a React component, to a Google Map. A Node server receives tweets from the Twitter streaming API, which are then piped to the client using socket.io. Still a work in progress but the app's most basic feature, getting data and displaying it, is working.

React Contact Manager

This is a very simple contact management tool which I made while I was learning how to use Facebook's React library. I also used Redux to manage the application's state.

Fruit Machine

I was inspired to make this after I watched a Javascript debugging tutorial in which the author had made a spinning wheel game to use as an example. All the logic for the app is written in plain JS.

Canvas Experiment

This demo is the first result of my experiments with the HTML5 Canvas API. Working with this technology is a lot of fun, and I will be starting a new Canvas project soon.

Music App

I prototyped this application for a creative agency in Bath, using the Last.fm API. They asked me to make something which would allow a user to search for an artist or group, and return a list of similar artists which the user could then explore.

tomaszpodpora.com

I was hired to design and build a responsive, bilingual site for a professional addiction therapist. The client wanted the site to be accessible to English and Polish speakers, so I wrote a program in Javascript which switches between the default English and the custom Polish translation. Wordpress was used as the CMS.

Metronome

I play guitar as a hobby and I had the idea of building a metronome in Javascript that I could practice with. Working on this personal project had the dual benefit of allowing me to improve my Javascript skills, and produce a tool that I use on a daily basis.

cardiffbaytours.co.uk

I designed and coded this website for a tour company based in Cardiff, using jQuery for the sliding menus and Photoshop for some of the graphics. Plugins were utilised for the photo gallery and contact form.
